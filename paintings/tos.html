pAIntings - AI Art Generator

How to Use

Enter a prompt (such as "a monkey eating a watermelon while jumping over a tree in a colorful tropical forest") and select an art style.
3 unique images will be generated each time you press the generate button.
Doing so uses up 1 credit, which you can stock up on using the "+" button in the top right corner.

Data Collection

pAIntings does not collect any type of data for tracking.
Usage data is collected throughout the app for analytics purposes. This happens anonymously and can't be linked to your identity.

Image Generation

Paintings are generated using the OpenAI API.

Content Moderation

pAIntings uses the OpenAI moderation API in order to comply with OpenAI's usage policies.
You, the user, remain responsible for all prompts entered into the pAIntings app and thus transfered to the OpenAI API.

Refund Policy

pAIntings is distributed through the Apple App Store, meaning that all payments and purchases are handled by Apple.
Thus, pAIntings is unable to to offer any kind of refunds. Apple has a refund program for In-App-Purchases.
